Typography for Weather Site

Heading Font

Bangers, because it's a very simple yet elegant and has a "I'm in charge" look about it and that it can't be messed with, just like nature.

Paragraph Font

Russo One. It is a very clean and readible font, easy on the eyes and just like nature is a very outspoken font that goes quite well with abrupt moods. Perfect as both a formal and a casual font.
